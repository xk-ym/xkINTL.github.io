    2.3 archlinux

    ArchLinux升级提示error: failed to update core (unable to lock database)的解决办法

    在使用ArchLinux的pacman命令做系统或软件升级时，有时候会碰到如下的报错，从而导致无法继续升级，甚至在kill相应的pacman进程也无效
    其实，这是Pacman数据库因之前的操作异常而被锁定，因此手动解锁即可。命令如下

    rm -f /var/lib/pacman/db.lcki
pacman -Syu

    2.4 Manjaro问题

    2.4.1 manjaro安装时卡住的解决办法

    Manjaro是Arch Linux的衍生发行版。拥有非常华丽的界面和极高的人气。拥有Arch的所有功能并且弥补了arch安装繁琐的不足。对于新手来说，提供了图形安装程序，预装桌面环境，和常用图形界面软件，真正的开箱即用。
    最新版本的Manjaro安装时可能遇到以下错误：

    1. Started TLP system startup/shutdown
    2. A start job is running for livemedia mhwd scripe(xx s/no limit)

    其实这两个错误都是因为显卡驱动的原因导致的.
    解决办法如下：
    引导安装盘或者镜像开机,看到启动菜单的时候,,用方向键移动光标，选中boot项，不要按回车，按E键编辑
    把driver=free改为driver=intel
    把下面的文字加入到driver=intel的后面，注意空格，
    xdriver=mesa acpi_osi=! acpi_osi=”Windows 2009”
    改好以后检查一下，最后效果如下所示，
    driver=intel xdriver=mesa acpi_osi=! acpi_osi=”Windows 2009″
    改好了以后就可以按F10启动了
